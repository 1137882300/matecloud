## 如果您觉得有帮助，请点右上角 "Star" 支持一下谢谢

MateCloud是一款基于Spring Cloud Alibaba的微服务架构。旨在为大家提供技术框架的基础能力的封装，减少开发工作，让您只关注业务。

### 功能特点
1.　采用最新的Spring Cloud Alibaba封装;

2.　与Dubbo与Feign集成，使用dubbo和feign切换非常方便;

3.　支持多数据库;

4.　集成消息中间件RocketMQ，使用异步处理逻辑;

5.　前端采用vue（待开发）;

6.　支持报文加密和验证机制（待开发）。
